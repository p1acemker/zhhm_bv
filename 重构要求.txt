【现有功能必须保持】
1) CLI:
- get <by1>：查询指定 by1 的货描
- list [limit]：列出所有 by1
- search "<text>"：向量搜索 by1
- preview <prefix>：预览前缀匹配候选
- import <by1> [strategy]：智能导入，strategy=most_references|highest_score|combined（默认 most_references）
- batch-import <by1>...：批量智能导入
- add <by1> "<edesc>"：添加货描，自动去重（by1存在则追加不重复；不存在则创建）
- update <by1> "<edesc>" [--append]：替换或追加
- delete <by1>：删除 by1

2) FastAPI:
- python api.py 启动服务
- 已有路由、方法、参数字段名保持一致（不得 breaking）
