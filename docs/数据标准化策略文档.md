# 蝶阀产品 EDesc 数据清洗与标准化策略

## 1. 数据概况

- **数据来源**: `edesc_by1_filtered.csv`
- **总记录数**: 2001 条
- **by1 唯一编码数**: 60 个
- **数据范围**: 蝶阀（Butterfly Valve）产品的英文描述文本

## 2. 数据现状与问题分析

### 2.1 描述格式不统一

当前 EDesc 存在三种主要描述风格：

| 风格 | 示例 | 占比 |
|------|------|------|
| 逗号分隔多字段 | `DI LUG WAFER BV,EPDM SEAT,SS316 DISC,HIGHER LEVER, 4"/DN100` | ~65% |
| 空格分隔紧凑型 | `Grv BFV BVG-1 21bar/300psi 2"/DN50 GG w/switches` | ~19% |
| 自然语言/混合型 | `DUCTILE IRON 3 ENCENTRIC FLANGE VALVE /DN150` | ~16% |

### 2.2 缩写/全称混用

| 概念 | 缩写 | 全称 | 出现频率 |
|------|------|------|----------|
| 蝶阀 | BV / BFV | Butterfly Valve | 全部 |
| 球铁 | DI | Ductile Iron | 1659 |
| 沟槽 | GRVD | Grooved | 491 |
| 法兰 | FLGD | Flanged | 57 |
| 三元乙丙 | EPDM | — | 1353 |
| 丁腈橡胶 | NBR | — | 122 |
| 氟橡胶 | VTON | Viton | 8 |
| 不锈钢 | SS316 / SS304 | Stainless Steel 316/304 | 291 |
| 碳钢 | CS | Carbon Steel | 84 |
| 铸钢 | WCB | — | 少量 |
| 铜材 | COP / BRZ | Copper / Bronze | 20 |

### 2.3 尺寸表达不一致

| 格式 | 示例 | 出现次数 |
|------|------|----------|
| 英寸 + DN | `4"/DN100` | 1131 |
| 仅英寸 | `4"` | 1784 |
| 英寸 + MM | `4"/114MM` | 546 |
| 分数英寸 | `21/2"` | 175 |
| 仅 DN | `DN100` | 部分混合 |

### 2.4 压力等级表达不一致

| 格式 | 示例 | 出现次数 |
|------|------|----------|
| PSI | `300PSI` | 204 |
| PN | `PN25` | 121 |
| Class | `CL125` | 24 |
| bar | `21bar` | 部分混合 |

### 2.5 特殊特征识别困难

- **消防用阀**（Fire Riser）：538 条，部分描述明确标注，部分隐含在型号中
- **开关反馈**（Tamper Switch）：66 条
- **偏心类型**：双偏心 46 条、三偏心 1 条
- **可锁定手柄**（Lockable Lever）：14 条

### 2.6 操作方式未明确

| 操作方式 | 明确标注数 | 未标注数 |
|----------|-----------|----------|
| LEVER（手柄） | 593 | — |
| GEAR（涡轮） | 493 | — |
| 未明确 | — | 915 |

## 3. 标准化字段定义

将 EDesc 解析为以下结构化字段：

```
{
    "connection_type":  "连接方式",
    "body_material":    "阀体材料",
    "disc_material":    "阀板材料",
    "seat_material":    "密封材料",
    "actuation":        "操作方式",
    "size_inch":        "口径(英寸)",
    "size_dn":          "口径(DN)",
    "pressure_class":   "压力等级",
    "special_features": "特殊特征",
    "valve_type":       "阀门类型(中线/单偏心/双偏心/三偏心)"
}
```

### 3.1 连接方式 (connection_type)

| 标准值 | 识别关键词 | 对应 by1 前缀规则 |
|--------|-----------|------------------|
| `WAFFER` | WAFER, Waf, Waf BFV | D71x |
| `LUG` | LUG, LUGGED | D71xL / D371xL |
| `LUG_WAFER` | LUG WAFER 同时出现 | D71xLV |
| `FLANGED` | FLGD, FLANGE, Flange | D34x / D41x |
| `GROOVED` | GRVD, GROOVED, Grv, GRV | D81x / XD381x |
| `THREADED` | THD, THREADED, BSP | XD311x |

### 3.2 阀体材料 (body_material)

| 标准值 | 识别关键词 |

## 4. by1 编码解码规则

by1 编码遵循蝶阀产品成套命名体系，结构如下：

```
[X] D [系列] [连接] [阀板/密封] [变体] [尺寸]
 │   │    │      │        │       │      │
 │   │    │      │        │       │      └─ 4=DN100, 见尺寸表
 │   │    │      │        │       └─ 附加特征(K=长颈, V=带齿轮箱, E=消防认证)
 │   │    │      │        └─ X=EPDM, S=特殊密封
 │   │    │      └─ L=LUG, 3=中线对夹, 4=法兰, 7=对夹, 8=沟槽
 │   │    └─ 3=双偏心, 7=中线, 8=沟槽中线
 │   └─ D=蝶阀
 └─ X=消防认证(FM/UL)
```

### 4.1 前缀含义对照

| 前缀 | 含义 |
|------|------|
| **D71x** | 中线对夹蝶阀 |
| **D81x** | 中线沟槽蝶阀 |
| **D371x** | 中线对夹蝶阀（大口径/长系列） |
| **D381x** | 中线沟槽蝶阀（大口径/长系列） |
| **D34x** | 双偏心法兰蝶阀 |
| **D41x** | 中线法兰蝶阀 |
| **D373x** | 双偏心对夹蝶阀 |
| **D343x** | 三偏心蝶阀 |
| **XD** | 消防认证系列 |

### 4.2 连接方式后缀

| 后缀 | 含义 | 示例 |
|------|------|------|
| 无 L | WAFFER 对夹 | D71X4 |
| L | LUG 凸耳 | D71XL4 |
| LV | LUG WAFFER 双用 | D71XLV4 |
| 无后缀 + 8x | GROOVED 沟槽 | D81X4 |

## 5. 数据清洗流程

### Step 1: 文本规范化

```
1. 统一转大写
2. 去除首尾空格、多余空格
3. 统一分隔符：全角逗号→半角逗号，中文标点→英文标点
4. 修正拼写错误：ENCENTRIC → ECCENTRIC, THEREADED → THREADED
5. 统一缩写：BUNA-N → NBR, FLANGE → FLGD (或反向展开)
```

### Step 2: 字段提取

按以下优先级策略提取字段：

**策略 A: 逗号分隔型描述**（占比最大）

```
DI LUG WAFER BV,EPDM SEAT,SS316 DISC,HIGHER LEVER, 4"/DN100
 ↓            ↓         ↓          ↓            ↓
 连接方式+阀体  密封材料   阀板材料    操作方式      口径
```

逐段按关键词匹配提取。

**策略 B: 空格分隔紧凑型描述**

```
Grv BFV BVG-1 21bar/300psi 2"/DN50 GG w/switches
 ↓             ↓            ↓          ↓     ↓
 连接类型      压力等级      口径      阀体  特殊特征
```

用正则表达式逐一匹配。

**策略 C: 自然语言型描述**

关键词搜索 + 正则匹配混合策略。

### Step 3: 缺失字段推断

对于缺失字段，按以下规则推断：

| 缺失字段 | 推断规则 |
|----------|---------|
| 操作方式 | 由 by1 后缀推断：V=GEAR，无标记=LEVER；消防系列默认 GEAR |
| 密封材料 | DI+EPDM DISC → 密封=EPDM；DI+NBR DISC → 密封=NBR |
| 阀体材料 | GRVD/GROOVED + 无材料说明 → DI |
| 连接方式 | 由 by1 前缀推断：D71x=WAFER, D81x=GROOVED, D34x=FLANGED |
| 阀门类型 | D34x → DOUBLE_ECCENTRIC, D7x/D8x → CENTER |

### Step 4: 数据验证

```
1. 口径一致性校验：inch 与 DN 值是否匹配标准对应表
2. 必填字段完整性检查：连接方式、阀体材料、口径不能为空
3. 逻辑一致性检查：
   - GROOVED 连接不应出现 DN > 300
   - 消防认证产品应标记 FIRE_RISER
   - 双偏心阀门不应出现 LEVER 操作
4. 与 by1 编码交叉验证：解析结果应与 by1 编码规则一致
```

## 6. 标准化输出格式

每条记录输出为 JSON 结构：

```json
{
    "original_edesc": "DI LUG WAFER BV,EPDM SEAT,SS316 DISC,HIGHER LEVER, 4\"/DN100",
    "by1": "D71XLV4",
    "parsed": {
        "valve_type": "CENTER",
        "connection_type": "LUG_WAFER",
        "body_material": "DI",
        "disc_material": "SS316",
        "seat_material": "EPDM",
        "actuation": "LEVER",
        "size_inch": 4,
        "size_dn": 100,
        "pressure_class": null,
        "special_features": []
    },
    "confidence": "HIGH"
}
```

### 置信度等级

| 等级 | 条件 |
|------|------|
| HIGH | 所有必填字段已明确提取，与 by1 编码一致 |
| MEDIUM | 部分字段需推断，但与 by1 编码一致 |
| LOW | 多个字段缺失且无法可靠推断 |
